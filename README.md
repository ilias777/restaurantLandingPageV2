## To try this website

Clone the repo
```shell
https://github.com/ilias777/restaurantLandingPageV2.git
```

Navigate to the folder
```shell
cd restaurantLandingPageV2
```

Install npm packages
```shell
npm install
```

Run dev server to see the page
```
npm run dev
```
